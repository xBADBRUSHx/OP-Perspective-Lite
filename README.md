# OP Perspective Lite

OP Perspective Lite 是 OnePunchPerspective 的透视辅助线精简版，用于 Adobe Photoshop。

## 下载

请从 GitHub Releases 下载最新安装包：

https://github.com/xBADBRUSHx/OP-Perspective-Lite/releases/latest

安装包文件名：

```text
OP-Perspective.exe
```

安装前请关闭 Photoshop。

## 功能范围

Lite 只保留透视辅助线工作流：

- 线性透视网格。
- 等距鱼眼透视网格。
- 立体鱼眼透视网格。
- 透视辅助线上的派生点和控制点。
- 用于调整透视辅助线参数的 WebView2 控制面板。

Lite 不包含 Object、Mesh、Transform 或场景编辑工具。

## Photoshop 版本支持

- Photoshop 2019-2020：使用 SDK CC 2019 bridge。
- Photoshop 2021-2023：使用 SDK 2023 bridge。
- Photoshop 2024 及以上：使用 SDK 2026 bridge。

## 更新提示

插件会检查本仓库的 `latest.json`。当有新版本可用时，Lite 控制面板标题右侧会显示更新提示按钮，点击后会打开 GitHub Releases 页面。
