## 功能范围

OP Perspective Lite 是只保留透视辅助线能力的精简版本。

- 线性透视网格。
- 等距鱼眼透视网格。
- 立体鱼眼透视网格。
- 透视辅助线上的派生点和控制点显示。
- WebView2 控制面板。

不包含 Object / Mesh / Transform / 场景编辑工具。

## 支持版本

- Photoshop 2019-2020：使用 Photoshop SDK CC 2019 bridge。
- Photoshop 2021-2023：使用 Photoshop SDK 2023 bridge。
- Photoshop 2024 及以上：使用 Photoshop SDK 2026 bridge。

安装前请关闭 Photoshop。

## 更新提示

当有新版本发布时，Lite 控制面板标题右侧会显示更新提示按钮。点击后会打开本仓库的 GitHub Releases 页面。
